02 : Works

I created a number of Python projects as a student of The App Brewery and Zero To Mastery. These include Ethical Hacking Tools, Games, GUIs, and Web Apps. You can see some of them below.

001/006

Backdoor

A backdoor program that works with a server program that lets the user control a target machine. The backdoor program can: send data & receive data, connect to a specific Kali Linux machine over a specific port number, upload & download files, and receive commands from the user. The server program can: send & receive data, upload & download files, and send user commands to the backdoor program on a target computer.

002/006

SnakeGame

A version of the Snake game. The player starts off with a small snake which grows longer as it "eats" food pellets that appear in random locations on the game screen. The player's objective is to "eat" food pellets to increase their score and lengthen their snake while avoiding collisions with the walls and segments of the snake's tail.

003/006

PongGame

A version of the classic game Pong. The game features a gold colored ball, a red-colored paddle on the left side of the screen that's controlled by the "w" and "s" keys, and a blue-colored paddle on the right side of the screen that's controlled by the "up" and "down" keys, and a scoreboard. The objective of the game is to be the first to get 5 points.

004/006

GUIQuizApp

A trivia quiz game GUI app that presents 10 true or false questions taken from the Open Trivia Database API. The user answers questions by clicking on a green "True" button or a red "False" button. The app then shows a flash of green or red depending on whether the user answered correctly. The app tracks of the number of correct answers through a scoreboard and lets the user know when there are no more questions.

005/006

FlashCardApp

A GUI app that presents flash cards with French and English words on them. French words are shown first. If the user doesn't know a word's meaning, the English translation will appear after three seconds. The user can click a red button if they don't know the answer and get a new French word card. If the user does know the word, clicking the green button will remove it from the list of French words and then present a new card.
